# StudentContacts
本程序为一个基于mfc和mysql的学生通讯录
程序使用mfc实现用户交互界面，使用mysql数据库保存通讯录信息
用户可以使用程序进行注册和登录，登录后可查询数据库中保存的学生信息，且用户可以修改自己的信息，
用户使用学号作为账号进行注册和登录。
